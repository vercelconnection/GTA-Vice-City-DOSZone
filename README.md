# GTA-Vice-City-DOSZone
chrome extension which allows to play GTA vice city on DOS Zone again.

### Step-by-step Installation

1. **Download or clone the extension folder**  
   Get 2 files (manifest.json and content.js)

2. **Open Chrome Extensions page**  
   Open Chrome and go to: chrome://extensions/

3. **Enable Developer mode**  
In the top-right corner, toggle the switch labeled **Developer mode** to ON.

4. **Load the unpacked extension**  
Click the **Load unpacked** button that appears in the top-left corner.

5. **Select the extension folder**  
Navigate to and select the folder containing your extension files (the one with `manifest.json` inside it).  
Click **Select Folder**.

6. **Done!**  
The extension should now appear in your list and be active.  
You can pin it to the toolbar by clicking the puzzle icon â†’ the pin icon next to your extension.

7. **Play GTA**
Go to (https://cdn.dos.zone/vcsky/release/vc-sky-en-v6/index.html?lang=en)[https://cdn.dos.zone/vcsky/release/vc-sky-en-v6/index.html?lang=en] and choose "gta-vc.exe" file as original file and play game.
